# Uzycie import yaml:

```
apiVersion: v1

metadata:
  name: biuro-prezesa
  namespace: krzak-pol-magazyn
spec:
  containers:
  - name: busybox
    image: busybox
    # pętla sleep kontener nie zakończy pracy
    command: ["sh", "-c", "while true; do sleep 3600; done"]

      mountPath: /dane_prezesa

  - name: dane-storage
    persistentVolumeClaim:
      claimName: faktury-pvc
```

Aby wykonać punkt nr 3

openssl req -x509 -nodes -days 365 ^
-newkey rsa:2048 ^
-keyout tls.key ^
-out tls.crt ^
-subj "/CN=nginx.193.187.69.235.nip.io/O=krzak-pol-web"

kubectl create secret tls nginx-tls --cert=tls.crt --key=tls.key -n krzak-pol-web
